Crystal Collector

Instruction:


You will begin with a random number at start.

There are 4 crystals with different hidden values. You click each crystal to add its hidden value to the sum. If that sum is equal the the random number, you win. If the sume is greater then the random number, you lose. The game restart after a win or lose with new random number and new hidden crystal values.

Click start to start the game
